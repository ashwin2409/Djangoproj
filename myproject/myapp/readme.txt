GET /clients/: Lists all clients.
POST /clients/: Creates a new client.
GET /clients/<id>/: Retrieves details for a specific client, including its projects.
PUT /clients/<id>/: Updates a client's information.
DELETE /clients/<id>/: Deletes a client.   http://127.0.0.1:8000/clients/2/
POST /projects/: Creates a new project.
GET /projects/assigned/: Lists projects assigned to the logged-in user.
